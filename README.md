## 🎯 What is Bingo Mixer?

Bingo Mixer transforms traditional networking and icebreakers into an engaging game! Perfect for:

- 🎓 **Conferences & meetups** - Help attendees connect meaningfully
- 🏢 **Team building events** - Break down barriers between colleagues  
- 🎉 **Social gatherings** - Turn awkward small talk into fun conversations
- 👥 **Workshops & classes** - Help participants find common ground

**How it works:** Each player gets a unique 5×5 bingo board filled with prompts like "Find someone who has lived in 3+ countries" or "Find someone who plays an instrument." Mingle, match prompts to people, and race to get 5 in a row!

---

## ✨ Features

- 🎲 **Randomized Boards** - Every player gets a unique board configuration
- 📱 **Mobile-First Design** - Perfect for on-the-go networking
- 🎨 **Beautiful UI** - Modern, polished interface with smooth animations
- ⚡ **Lightning Fast** - Built with Vite for instant hot-reload development
- 🎭 **Customizable Prompts** - Easy to adapt for different events and audiences
- 🏆 **Winner Celebration** - Fun confetti animation when someone wins
- 🌐 **No Backend Required** - Runs entirely in the browser

## 🚀 Quick Start

### Prerequisites

- [Node.js 22+](https://nodejs.org/) or higher

### Local Development

```bash
# Clone the repository
git clone https://github.com/sgraubard/my-bingo-mixer.git
cd my-bingo-mixer

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be running at `http://localhost:5173` 🎉

### Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist/` folder.
